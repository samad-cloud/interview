**FunnelCard sub-component** (internal, not exported):
- `<button>` element with `onClick={() => onStageClick(FUNNEL_TO_STAGE_FILTER[key])}`
- Tailwind classes: `'relative flex-1 rounded-lg border p-3 text-left transition-colors cursor-pointer'`
- When `isSelected` (activeStage === FUNNEL_TO_STAGE_FILTER[key]): `'border-[#6366F1] bg-[rgba(99,102,241,0.06)]'`
- When not selected: `'border-[#1E293B] bg-[#0F172A] hover:border-[#2D3F55] hover:bg-[#1A2332]'`
- Conversion % pill (absolute top-2 right-2): show only when `prev` is not null AND `prev > 0`. Formula: `Math.round((count / prev) * 100)`. Style with `background: \`${color}26\`` and `color: color`. Text: `{pct}%`, class `text-[9px] font-semibold px-1.5 py-0.5 rounded`.
- Count: `<div className="text-[22px] font-bold tabular-nums leading-tight" style={{ color }}>{count.toLocaleString()}</div>`
- Label: `<div className="text-[10px] text-[#6B7280] mt-0.5 whitespace-nowrap">{label}</div>`

**FunnelRow layout**:
```tsx
<div className="flex items-stretch gap-0">
  {FUNNEL_STAGES.map((stage, i) => (
    <>
      {i > 0 && (
        <div className="flex items-center px-1.5 text-[#4B5563] text-sm flex-shrink-0">→</div>
      )}
      <FunnelCard
        key={stage.key}
        label={stage.label}
        color={stage.color}
        count={stage.getStat(stats)}
        conversionPct={...}
        isSelected={activeStage === FUNNEL_TO_STAGE_FILTER[stage.key]}
        onClick={() => onStageClick(FUNNEL_TO_STAGE_FILTER[stage.key])}
      />
    </>
  ))}
</div>
```

Use `React.Fragment` with `key` for the map wrapper since it contains two sibling elements.

Do NOT use dynamic Tailwind class construction for colors (e.g., no template literals in className). Use `style={{ background: ... }}` for gradient/color values, and `style={{ color: ... }}` for the count number and pill text — this is mandatory to prevent Tailwind purge in production.
